→node.js

Install node.js.

node.js 中自带了一个叫做软件包管理器的工具 npm。

→webpack

通过 npm 安装 WebPack：

$ npm install webpack -g

webpack 现在已经全局安装成功，并且在命令行中可以直接使用 webpack 命令。

→在项目中使用 WebPack

最好让webpack也作为你的项目中的依赖，这样你就可以选择一个本地的webpack版本，而不会被迫使用单一的全局版本。

为npm添加一个 package.json 配置文件：

$ npm init

如果你不想发布到 npm ，可忽略此问题

在 package.json 中安装和添加 webapck:

$ npm install webpack --save-dev

→版本

有两个版本的 webpack 可用：一个稳定版和 beta 版本。

  • beta版会在版本号中增加 -beta 字符串予以区分
  • beta版包含无价值的变化或实验性的功能，且测试较少
  • 版本差异，请参阅更新日志

对于正式的项目建议使用稳定版本：

$ npm install webpack@1.2.x --save-dev

→开发工具

如果你想使用开发工具，按照下面的方法进行安装：

$ npm install webpack-dev-server --save-dev
